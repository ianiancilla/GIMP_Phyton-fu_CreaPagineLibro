Per usare lo script:

Apri GIMP, EDIT > PREFERENCES > FOLDERS > PLUG_INS
Seleziona e copia il path dei plug-in personali (quello nel tuo user folder)
Chiudi GIMP e apri il filder di cui hai copiato il path

Incolla il file .py in quel folder

Apri GIMP
Apri il file .XCF che trovi nel repo
Selezione dalla barra in alto ScriptPerIvana e lo script
Inserisci i dati richiesti
Avvia =)
